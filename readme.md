2. Run the project
3. Send the list of the emails as Post request to http://localhost:8080/emails/validate

Example:

 Sample payload 
 ```
   [
        "samikshyagautam@gmail.com", 
        "samikshya.gautam+sam@gmail.com",
        "samikshyagautam+5@gmail.com",
        "Samikshya.gautam@gmail.com"
   ]
 ```
